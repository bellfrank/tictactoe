    Flashcards

    Implemented using REACT which is a declarative style of coding (this is what I want it to look like, you figure it out).

    It uses JSX instead of JavaScript. I also use Babel to translate JSX -> JavaScript since most browsers don't understand JSX.

    In REACT we have to think about our application in components and states, and how those components are going to behave.

    What makes REACT powerful is if you change the state, the view also changes.

    Known Bugs:

    - Currently working on a feature using componentDidMount that allows us to store data in browsers when user exits or refreshes page. I need to reconsider my approach since browsers tend to store information in Local Storage as a string and need to parse and stringify my data acoordingly.
